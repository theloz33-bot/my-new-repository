# Project Blueprint: 정치 시사 블로그

## 1. 개요

이 문서는 웹 컴포넌트, 최신 CSS 및 순수 JavaScript를 사용하여 현대적이고 반응이 빠르며 유지 관리가 쉬운 블로그를 만들기 위한 계획을 설명합니다.

## 2. 구현된 기능

*   **웹 컴포넌트 기반 블로그 게시물:** 개별 블로그 게시물을 표시하기 위해 재사용 가능한 `<blog-post>` 웹 컴포넌트를 만들었습니다.
*   **방명록 기능:** 사용자가 이름과 메시지를 남길 수 있는 `<guest-book>` 웹 컴포넌트를 추가했습니다. 데이터는 브라우저의 로컬 저장소에 저장됩니다.
*   **동적 콘텐츠 렌더링:** `main.js`에서 정치/시사 관련 주제의 블로그 데이터를 동적으로 렌더링합니다.
*   **모던 디자인 시스템:** CSS 변수를 활용하여 색상, 타이포그래피, 레이아웃 등 일관된 디자인 시스템을 적용했습니다.

## 3. 파일 구조

```
.
├── firebase.json
├── .firebaserc
├── blueprint.md
├── index.html
├── style.css

```

## 4. 현재 계획: 콘텐츠 추가 및 방명록 페이지 분리

블로그 콘텐츠를 보강하고, 메인 페이지와 방명록 페이지를 분리하여 서비스 구조를 개선합니다.

### 4.1. 목표

*   **콘텐츠 확장:** 더 많은 뉴스 스크랩을 추가하여 블로그를 풍성하게 만듭니다.
*   **페이지 분리:** 방명록 기능을 별도의 페이지로 분리하여 각 페이지의 역할을 명확히 합니다.
*   **내비게이션 구현:** 사용자가 메인 페이지와 방명록 페이지를 쉽게 오갈 수 있도록 내비게이션을 추가합니다.

### 4.2. 상세 실행 계획

1.  **뉴스 콘텐츠 추가 (`main.js`):**
    *   `posts` 배열에 사회, 경제 등 다양한 주제의 뉴스 기사 객체를 3개 이상 추가합니다.

2.  **방명록 페이지 생성:**
    *   **`guestbook.html` 생성:** 메인 페이지(`index.html`)와 유사한 구조를 가지되, `<main>` 영역에 `<guest-book>` 컴포넌트만 포함하는 새로운 HTML 파일을 생성합니다.
    *   **`guestbook-page.js` 생성:** `guestbook.html`에서 사용할 스크립트 파일로, `<guest-book>` 웹 컴포넌트를 import하는 역할을 합니다.

3.  **내비게이션 구현:**
    *   **`style.css` 수정:** 헤더 영역에 추가될 내비게이션 메뉴의 스타일(링크, 배치 등)을 정의합니다.
    *   **`index.html` 수정:** 헤더에 방명록 페이지로 이동하는 링크를 추가합니다.
    *   **`guestbook.html` 수정:** 헤더에 메인 페이지로 돌아가는 링크를 추가합니다.

4.  **기존 코드 리팩토링:**
    *   **`index.html` 수정:** 기존에 있던 방명록 컨테이너(`id="guestbook-container"`)를 삭제합니다.
    *   **`main.js` 수정:** `<guest-book>` 컴포넌트 import 및 관련 코드를 모두 삭제하여 블로그 게시물 렌더링에만 집중하도록 합니다.

5.  **최종 검토 및 배포:**
    *   모든 변경 사항이 의도대로 적용되었는지 확인하고 페이지 간 이동이 잘 되는지 테스트합니다.
    *   개선된 버전의 블로그를 Firebase에 다시 배포합니다.
