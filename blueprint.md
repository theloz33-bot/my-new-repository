# Project Blueprint: 코리안 뉴 데스크\n\n## 1. 개요\n\n이 문서는 웹 컴포넌트, 최신 CSS 및 순수 JavaScript를 사용하여 현대적이고 반응이 빠르며 유지 관리가 쉬운 뉴스 블로그를 만들기 위한 계획을 설명합니다.\n\n## 2. 구현된 기능\n\n*   **웹 컴포넌트 기반 UI:** 재사용 가능한 웹 컴포넌트(\`<blog-post>\`, \`<guest-book>\`)를 활용하여 UI를 모듈화했습니다.\n*   **페이지 분리:** 블로그 홈, 방명록, 공지사항, 기사 상세 보기 등 주요 기능을 별도의 페이지로 분리하여 구조를 명확히 했습니다.\n*   **동적 콘텐츠 렌더링:** 중앙 데이터 파일(`data.js`)로부터 데이터를 불러와 각 페이지의 콘텐츠를 동적으로 생성합니다.\n*   **내비게이션:** 모든 페이지에 일관된 헤더와 내비게이션 메뉴를 제공하여 사용자 이동성을 높였습니다.\n\n## 3. 파일 구조\n\n```\n.\n├── firebase.json\n├── .firebaserc\n├── blueprint.md\n├── index.html\n├── guestbook.html\n├── announcements.html\n├── article.html\n├── style.css\n├── main.js\n├── guestbook-page.js\n├── announcements-page.js\n├── article-page.js\n├── data.js\n└── components\n    ├── blog-post.js\n    └── guest-book.js\n```\n\n## 4. 현재 계획: 타이틀 변경 및 스타일 수정\n\n블로그의 브랜딩을 강화하고 시각적 매력을 높이기 위해 타이틀과 색상을 변경합니다.\n\n### 4.1. 목표\n\n*   **타이틀 변경:** 블로그의 이름을 "코리안 뉴 데스크"로 변경합니다.\n*   **스타일 업데이트:** 새로운 타이틀에 맞춰 헤더의 `<h1>` 태그 색상을 시각적으로 돋보이는 색으로 변경합니다.\n\n### 4.2. 상세 실행 계획\n\n1.  **스타일 수정 (`style.css`):**\n    *   `header h1` 선택자의 `color` 속성을 신뢰감을 주는 짙은 네이비 색상 (`#003366`)으로 변경합니다.\n\n2.  **HTML 파일 업데이트:**\n    *   모든 HTML 파일 (`index.html`, `guestbook.html`, `announcements.html`, `article.html`)의 `<title>` 태그와 `<h1>` 태그 내용을 "코리안 뉴 데스크"로 일괄 변경합니다.\n
3.  **스크립트 파일 업데이트 (`article-page.js`):**\n    *   기사 상세 페이지의 타이틀을 동적으로 설정하는 부분에서 블로그 이름을 "코리안 뉴 데스크"로 수정합니다.\n
4.  **최종 검토 및 배포:**\n    *   모든 페이지에서 타이틀과 색상이 올바르게 적용되었는지 확인하고, 최종 버전을 배포합니다.\n
